# math.js

A work-in-progress library for solving anything mathematical.

## Setup

```
$ git clone https://github.com/HegartyCalculator/math.js.git math
or
$ gh repo clone HegartyCalculator/math.js math
```

```
$ cd math
```

```
$ npm install
```

## Build

### Web

```
$ make build
```

### Minify

```
$ make minify
```
